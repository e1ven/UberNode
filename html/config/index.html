    console
  • Enable TMCIFreenet can optionally enable a text-mode telnet interface for retrieving keys, and managing your node. Some third party tools require this to be enabled.
  • Enable on stdout/stdin?If Enabled, the text mode interface runs in the same shell/command window in which you started freenet. If set to disabled, the TMCI server listens for telnet connections
  • IP address to bind toThe IP address that Freenet should use. This should usually be 127.0.0.1, if you aren't connecting to this freenet server from multiple machines. If you are, set it to the IP address of this server.
  • Telnet portThe port that the telnet server runs on
  • Allowed hostsA list of IP addresses that are allowed to connect to the telnet server. This may be either a single entry, or a comma-separated list.

    node.testnet
  • Enable testnet mode? (DANGEROUS)Testnet is an optional mode to run freenet in, which allows the developers full access to what you are doing. Having a few people, who are working with the developers, set this is a good idea, but most people should select Disabled. Enabling Testnet will stop your machine from connecting to most of freenet.

    node.updater
  • Automatically download new versionsIf enabled, Freenet will automatically detect and download new versions of itself. They will not automatically install unless you enable Automatically Install new Versions, below.
  • Automatically install new versions?If this setting, and "Automatically download new versions" (above) are enabled, freenet will automatically search for, and install new versions of the freenet software. It is HIGHLY recommended that you enable these features. As Freenet is still alpha software, new versions are released on occasion, which will not talk to older versions of freenet. If you set this to disabled, you may lose access to the network, if you do not keep up with updates.
  • Where should the node look for revocation ?Where should the node check, to see if the new version that was put out doesn't work properly, or shouldn't be trusted?
